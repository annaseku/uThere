# uThere Web App
This is a non-invasive location tracking web app, soon to be iOS. Create groups and add your important addresses to notify circles of arrivals and departures. Has native browser location tracking and uses Maps API to configure addresses.

This app was built using Lovable and can be accessed via https://uthere.lovable.app/.

Please feel free to make an account and use the app! Or else, have a look at the following demo pictures to get an idea.
